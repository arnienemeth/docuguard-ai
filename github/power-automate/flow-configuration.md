# Power Automate Flows — Configuration Guide

## Flow 1: DocuGuard - Send Anomaly Alert Email

### Overview
Sends a professionally formatted HTML email alert when the Copilot Studio chatbot detects or is asked about an anomaly.

### Setup Steps

1. **Create flow:** Power Automate → New → Instant cloud flow
2. **Trigger:** "When an agent calls the flow" (Skills)
3. **Name:** `DocuGuard - Send Anomaly Alert Email`

### Input Parameters

| Type | Name | Description |
|------|------|-------------|
| Text | DocumentId | Document identifier (e.g., INV-2026-007) |
| Text | AlertTitle | Title of the alert (e.g., "Critical Anomaly Detected") |
| Text | AlertMessage | Detailed description of the anomaly |
| Text | Severity | Alert severity level (Critical, High, Medium, Low) |
| Text | RecipientEmail | Email address to send the alert to |

### Actions

**Action 1: Send an email (V2)** (Office 365 Outlook)
- **To:** `RecipientEmail` (dynamic content)
- **Subject:** `[DocuGuard AI] {Severity}: {AlertTitle}`
- **Body (HTML):**

```html
<div style="font-family: Segoe UI, Arial, sans-serif; max-width: 600px;">
  <div style="background-color: #1B2A4A; padding: 20px; color: white;">
    <h2 style="margin:0;">⚠️ DocuGuard AI - Anomaly Alert</h2>
  </div>
  <div style="background-color: #FFF4CE; border-left: 4px solid #D83B01; padding: 15px; margin: 15px 0;">
    <strong>Severity: </strong>{Severity}<br/>
    <strong>Document: </strong>{DocumentId}
  </div>
  <div style="padding: 15px;">
    <h3 style="color: #2B579A;">{AlertTitle}</h3>
    <p>{AlertMessage}</p>
  </div>
  <div style="background-color: #F2F2F2; padding: 15px; font-size: 12px; color: #666;">
    This alert was generated by DocuGuard AI Anomaly Detection System.<br/>
    Please review and take appropriate action.
  </div>
</div>
```

**Action 2: Respond to the agent**
- Output (Text): `Result` = `Alert email sent successfully to {RecipientEmail}`

## Flow 2: DocuGuard - Send Slack Alert

### Overview
Posts a formatted alert message to a Slack channel when anomalies require team attention.

### Setup Steps

1. **Create flow:** Power Automate → New → Instant cloud flow
2. **Trigger:** "When an agent calls the flow" (Skills)
3. **Name:** `DocuGuard - Send Slack Alert`

### Input Parameters

| Type | Name | Description |
|------|------|-------------|
| Text | DocumentId | Document identifier |
| Text | AlertTitle | Title of the alert |
| Text | AlertMessage | Detailed anomaly description |
| Text | Severity | Alert severity level |

### Actions

**Action 1: Post message (V2)** (Slack)
- **Channel:** `#all-docuguard-alerts`
- **Message Text:**

```
🚨 *DocuGuard AI - Anomaly Alert*

*Severity:* {Severity}
*Document:* {DocumentId}

*{AlertTitle}*

{AlertMessage}

_Take action in the DocuGuard portal or reply to discuss._
```

**Action 2: Respond to the agent**
- Output (Text): `Result` = `Slack alert posted successfully`

### Slack Setup
1. Create a Slack workspace (or use existing)
2. Create a channel named `#all-docuguard-alerts`
3. Authorize the Slack connector in Power Automate during flow creation

---

## Connecting Flows to Copilot Studio

1. Open Copilot Studio → DocuGuard AI agent
2. Go to **Tools** tab
3. Click **"+ Add a tool"**
4. Select the **"Flow"** tab
5. Search for each flow name and add it
6. The chatbot can now call these flows during conversations
